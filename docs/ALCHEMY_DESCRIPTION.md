# Descrição para Alchemy

## Versão Curta (1 frase)

**"Building a decentralized performance protocol that transforms physical effort into verifiable digital value using blockchain and IPFS."**

---

## Versão Média (2-3 frases)

**"We're building a decentralized fitness platform where athletes can prove their performance, compete in challenges, and earn tokens. Using Account Abstraction for seamless UX, IPFS for permanent storage, and Polygon for low-cost transactions. The protocol enables trustless validation and automatic reward distribution."**

---

## Versão Completa (1 parágrafo)

**"WOD [X] PRO (wod.eth) is a decentralized platform that transforms physical performance into permanent, verifiable digital assets. Athletes submit video proof of effort, compete in on-chain challenges with entry fees, and earn $WOD tokens through community validation. We use Alchemy Account Kit for frictionless social login, Lighthouse/NFT.Storage for permanent IPFS storage, and Polygon for cost-effective smart contract execution. The protocol eliminates intermediaries by using smart contracts for automatic reward distribution based on decentralized consensus."**

---

## Pontos-Chave para Mencionar

✅ **Account Abstraction** - Social login (Google/email) com smart wallets
✅ **IPFS Storage** - Permanent proof storage (Lighthouse/NFT.Storage)
✅ **Polygon L2** - Low gas costs
✅ **Trustless** - No central authority needed
✅ **Community Validation** - Decentralized consensus
✅ **Automatic Rewards** - Smart contract distribution

---

## Use Cases para Alchemy

- **Account Kit**: Social login para mass adoption
- **Pay**: On-ramp PIX (Brasil) para fiat → token
- **RPC**: Reliable node infrastructure for Polygon
- **Gas Manager**: Potential subsidy for user onboarding

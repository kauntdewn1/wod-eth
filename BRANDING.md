# Branding - WOD [X] PRO

## Nomenclatura Oficial

### 🎯 Nome do Projeto
**WOD [X] PRO**

- Nome para apresentação, marketing e público
- Usado em interfaces, documentação de usuário e pitch

### 🌐 Domain ENS
**wod.eth**

- Domain na rede Ethereum (ENS)
- Identidade on-chain do protocolo
- Pode ser usado como wallet address

### 💰 Token
**WOD**

- Ticker: `WOD`
- Símbolo: `$WOD`
- Nome completo: WOD Token
